这是我的第一篇博客！这里记录一下使用hexo-admin来管理并发表博文的过程.了解更多请关注我的Github. 正文安装插件1$ npm install --save hexo-admin 插件名称hexo-admin 启动hexo-admin1$ hexo server -d 打开 http://localhost:4000/admin 就可以访问到hexo-admin管理页面了. 发布并部署到网站上然后，Deploy之前，还需要编辑配置文件_config.yml。(否则会出现Error: Config value “admin.deployCommand” not found或者Error: spawn hexo ENOENT之类的报错。)如果是Windows则在末尾加上一个批处理文件 12$ admin: deployCommand: 'hexo-publish.bat' 然后在同级目录新建hexo-pubish.bat文件，文件内容如下： 1$ hexo g -d 然后就可以直接Deploy了!
